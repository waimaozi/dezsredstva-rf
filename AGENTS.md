# Дезинфицирующиесредства.рф — Блог о дезинфекции

## Описание
Независимый информационный блог о дезинфицирующих средствах и дезинфекции. Переводные статьи мировых исследований, обзоры практик, полезные инструменты. Никаких продаж, никаких каталогов продуктов.

## Стек
- Astro + Tailwind CSS (SSG)
- Деплой: GitHub Pages
- Язык контента: русский

## Структура
- `/` — Главная: описание блога + лента последних статей
- `/articles/[slug]` — Статьи (Markdown/MDX в src/content/articles/)
- `/tools/` — Калькулятор разведения растворов

## Контент-модель
Статьи хранятся как .md файлы в src/content/articles/ с frontmatter:
```yaml
title: "Заголовок"
description: "Описание для SEO"
date: 2026-03-15
tags: ["исследования", "патогены"]
source: "https://pubmed.ncbi.nlm.nih.gov/..." # оригинал
```

## Дизайн
- Чистый, профессиональный, медицинская тематика
- Белый фон, синий/голубой акцент
- Мобильная адаптация
- Шрифт с поддержкой кириллицы

## SEO
- Meta title, description на каждой странице
- Sitemap.xml, robots.txt
- Schema.org (Article, Blog)
- Быстрая загрузка (статика)

## Спонсор
Единственное упоминание Химитек — строка в футере: "При поддержке chemitech.ru"
НЕТ каталогов, НЕТ продуктовых карточек, НЕТ упоминаний в контенте.
