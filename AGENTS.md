# Дезинфицирующиесредства.рф — Агрегатор дезинфицирующих средств

## Описание проекта
Информационный сайт-агрегатор по дезинфицирующим средствам. SEO-сателлит для Химитек (chemitech.ru).

## Стек
- Astro + Tailwind CSS
- Статическая генерация (SSG)
- Деплой: GitHub Pages
- Язык контента: русский

## Структура сайта
- `/` — главная (агрегатор, обзор)
- `/products/` — каталог дезсредств (карточки с описанием, концентрации, применение)
- `/research/` — дайджесты исследований (автопубликация через N8N)
- `/regulations/` — СанПиНы, ГОСТы, нормативные документы
- `/tools/` — калькулятор разведения, генератор графика дезинфекции
- `/compare/` — сравнительные таблицы средств

## SEO-требования
- Домен: дезинфицирующиесредства.рф (130к запросов/мес)
- Каждая страница = отдельный кластер запросов
- Meta title, description, OG-теги на каждой странице
- Sitemap.xml, robots.txt
- Семантическая HTML-разметка (schema.org для Product, Article)
- Быстрая загрузка (< 1сек, статика)

## Дизайн
- Профессиональный, чистый, медицинская тематика
- Цвета: белый фон, синий/голубой акцент (ассоциация с чистотой/медициной)
- Мобильная адаптация обязательна
- Шрифт: Inter или аналог (поддержка кириллицы)

## Контент-план (стартовый)
1. Главная — обзор, навигация по разделам
2. 5 карточек дезсредств Химитек
3. Калькулятор разведения растворов
4. 3 справочные статьи (СанПиН, сравнение хлорных/бесхлорных, выбор средства для общепита)
